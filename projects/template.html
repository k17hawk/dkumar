Template for deep learning

  • Tech Stack: Python,Flask, DVC, tox, shell script, YAML, cfg, toml
  • Github URL: Project Link

This project contains ready to use folder structure having all the necessary folders and pipeline tools implemented. All the folders are created in Template.py file having all the name of files also have it's logger and exception handler. The setup.py file provides the version to project and make entire project as a package. Setting up the conda environment and required python version with keep installing required package is time consuming, so i have created bash file for handling all of this sub task.
