Insurance Fraud Detection

  • Tech Stack: Python,Pandas, Numpy, Matplotlib, Sklearn, Machine Learning,Classification, Boosting techniques
  • Github URL: Project Link

For this project, after analysing the data and complete selection of feature the first model trained was, support vector machine (SVM) with the accuracy of 75%, after hypertuning the parameter with cross fold of 5, an accuracy reached to around 76%. Other models which was trained were GradientBoosting classifier,XGBoost Classifier But because of unbalanced between fraud or geniun value there was bias in prediction and usually model predicted fraud.
