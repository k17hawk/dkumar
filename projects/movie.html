Hollywood movie revenue predition

  • Tech Stack: Python,Flask, Machine Learning, Model Factory, BeautifulSoup, tox, shell script, YAML, cfg, toml, CI/CD, Pipelining, Anaconda,
  • Github URL: Project Link

This project is created using the template from Deep Learning. The dataset are scraped from box office mojo and also downloaded from IMDB website. The files are merged using based on their name, but first it is lower cased and space are removed. I begin with creating entity having class and object. There are six entity in this project which are: 1 data ingestion 2 data validation 3 data transformation 4 model training 5 model evaluation 6 Model push cloud The properities for entity are written in config.
